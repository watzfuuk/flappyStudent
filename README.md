##  Як запустити гру

### Для гравців (Викладача):
Готові скомпільовані файли (.apk для Android та .exe для Windows) можна завантажити у розділі **Releases**:
 **[Завантажити останню версію](https://github.com/watzfuuk/flappyStudent/releases)**

### Для розробників (Відкрити код):
1.  Переконайтеся, що у вас встановлено **Unity Hub** та **Unity 2022.3 LTS**.
2.  Клонуйте репозиторій:
    ```bash
    git clone https://github.com/watzfuuk/flappyStudent.git
    ```
3.  Відкрийте Unity Hub -> `Add` -> Виберіть папку з проектом.
4.  Запустіть проект (сцена `Scenes/MenuScene`).
